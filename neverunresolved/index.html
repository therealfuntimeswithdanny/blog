Don't Leave Domains Unresolved

Keep your domain always pointing to a live website with these simple solutions!

Option 1: Redirect via Cloudflare Page Rules

  1. Log in to your Cloudflare Dashboard and select your domain.
  2. Navigate to the Page Rules tab.
  3. Click on Create Page Rule.
  4. Enter this URL pattern that matches your domain: *yourdomain.com/*
  5. Under settings, choose Forwarding URL and select 301 â€“ Permanent Redirect.
  6. Set the destination URL to: https://madebydanny.uk/resolved
  7. Save and deploy your Page Rule.

This method configures a server-level redirect that is robust, SEO-friendly, and works regardless of client-side settings.

Option 2: Redirect via an External JavaScript File

  1. In your site's HTML file, add the following one line of code to the <head> section. 
<script src="https://madebydanny.uk/neverunresolved/api/v1.js"></script>

This client-side method automatically captures the current URL, encodes it, and appends it as a query parameter before redirecting the visitor.
