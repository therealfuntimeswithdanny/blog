Matter’s latest update adds tap-to-pair setup via NFC

Two groundbreaking updates are being introduced to Matter, designed to streamline the installation process of smart home devices. These new features include Multi-Device QR Codes and the tap-to-pair setup using NFC technology.

These updates are incorporated into the latest 1.4.1 version of Matter, the organization that oversees the smart home interoperability standard, collaboratively developed by Apple, Amazon, Google, Samsung, among others.

The Multi-Device QR Codes feature enables users to set up multiple gadgets bundled together, such as sets of 2, 4, or 6. For example, if I buy a pack of four smart light bulbs, instead of scanning each QR code individually, a single scan will connect all four at once. Although light bulbs are just an example, this capability applies to all devices. Tap-To-Pair, which is possibly my preferred feature, functions similarly to tap-to-pay systems on iPhones and Android devices. By leveraging NFC, users can skip the QR code entirely, merely tapping or bringing the phone close to a device to link it to a Matter platform. This approach speeds up the pairing process and is especially useful for devices with QR codes in awkward spots, like lightbulbs that need to be powered on for pairing.
